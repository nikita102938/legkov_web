Понятие квадратного уравнения
  • Квадратными уравнениями называют уравнения, записанные в виде: ax² + bx + c = 0
  • Здесь x играет роль неизвестного, a и b являются коэффициентами, c определяется как свободный член.
  • Полным квадратным уравнением является такое уравнение, которое имеет коэффициенты a, b и c отличные от нуля.
  • Приведенное квадратное уравнение — уравнение с коэффициентом a, равным единице, то есть вида: x² + bx + c = 0
  • Неполное квадратное уравнение — уравнение с нулевым коэффициентом b и/или нулевым свободным членом c вида: ax² + c = 0. Здесь a и c отличны от нуля.
    • Способы решения квадратного уравнения
  • I способ. Алгоритм решения через дискриминант
      • Решение квадратных уравнений с помощью дискриминанта не вызывает проблем, если следовать стандартному алгоритму и запомнить несколько формул. Порядок действий следующий:
      1. Приведение уравнения к стандартному виду: ax² + bx + c = 0. В том случае, когда уравнение изначально записано в стандартном виде, этот этап можно пропустить. Важно выполнить правильно определение коэффициентов a и b, а также свободного члена c. Например, x² + 2x - 3 = 0, где a=1, b=2, c=-3.
      2. Далее следует рассчитать дискриминант, используя формулу: D = b² - 4ac. Например, D = 2² - 4*1*(-3) = 4 + 12 = 16.
      3. Вычеслить корни уравнения с помощью формулы: x = -b ± √D
        2a                 Возможно несколько вариантов решения:
        • При D>0 уравнение обладает парой корней.
        • При D=0 формула сокращается до вида: x= -b/2a . В результате у такого уравнения имеется единственный корень.
        • При D < 0 исключается возможность извлечения корня из дискриминанта на третьем шаге алгоритма. В итоге уравнение не обладает решениями.
  • II способ. Решение через теорему Виета
      • Теорема Виета: корни приведенного квадратного уравнения, при a=1, в сумме составляют -b. Произведение корней равно c. Таким образом: x₁ + x₂ = -b; x₁ * x₂ = c.
      Поэтапный алгоритм действий при решении квадратных уравнений по теореме Виета:
    1. определение пары чисел, произведение которых дает свободный член уравнения, а в сумме они равны второму коэффициенту, записанному с противоположным знаком;
    2. вычисление корней уравнения.
  • III способ. Решение неполных квадратных уравнений
      • При b=0 и c=0: ax² = 0, x² = 0, x=0.
      • При b=0 и c≠0: ax² + c = 0, ax² = -c, x² = -c/a, x₁₂ = ±√(-c/a)
      • При b≠0 и c=0: ax² + bx = 0, x(ax + b) = 0, x=0 или ax + b = 0, x₁=0 x₂ = -b/a.
  • IV способ. Разложение квадратного трёхчлена на линейные множители.
      • Если трёхчлен вида ax²+bx+c(a≠0) удастся каким-либо образом представить в качестве произведения линейных множителей (kx + m)(lx + n) = 0, то можно найти корни уравнения, ими будут -(m/k) и -(n/l) действительно, ведь (kx+m)(lx+n)=0 <=> (kx+m=0, lx+n=0) а решив указанные линейные уравнения, получим вышеописанное. Квадратный трёхчлен не всегда раскладывается на линейные множители с действительными коэффициентами: это возможно, если соответствующее ему уравнение имеет действительные корни. Одним из частных случаев является квадрат суммы: (ax)²+2abx+b²=(ax+b)², (ax+b)²=0, x= -(b/a)
    Нестандартные уравнения
  • Метод через замену
      • Метод замены переменной - это один из основных методов решения сложных уравнений. Цель метода свести сложное уравнение к более простому путем введения новой переменной.
      Итак, находим одинаковые части уравнения, содержащие переменную x: это могут быть просто xⁿ, или целые выражения, зависящие от x, и обозначаем их новой переменной t(не имеет значения какой). Главное, чтобы после замены в исходном уравнении не осталось переменной x.
    • Биквадратные уравнения
      • Биквадратные уравнения внешне очень похожи на обыкновенные квадратные уравнения, только степень у переменной x вдвое больше: ax⁴+bx²+c=0; Где a, b и c всё те же коэффициенты из обычного уравнения. Чтобы решить такое уравнение, обозначим за переменную t = x². Тогда логично предположить, что x⁴ = (x²)² = t². Обратите внимание, что t ≥ 0,потому что квадрат всегда неотрицателен. Поэтому если у вас получатся отрицательные t, их надо будет выкинуть. Уравнение примет вид: at²+bt+c=0. Так наше уравнение четвертой степени превратилось в обыкновенное квадратное уравнение, которое можно решить через дискриминант. После того, как мы найдем корни t₁ и t₂, нужно будет вернуться к переменной x, ведь нас просят найти именно x, а не t.
    • Замена многочлена
      • Можно существенно упростить уравнение или неравенство, если в нем есть «куски» из одинаковых многочленов. При этом важно помнить: после замены в уравнении не должно оставаться исходной переменной x. Оно имеет вид: a*P²(x)+b*P(x)+c=0. где P(x) - это многочлен n−й степени; a, b - некоторые числа; Пусть t = P(x), тогда исходное уравнение принимает вид: at²+bt+c=0.
    • Другие случаи
      • Заменять можно практически все: квадратный корень, дроби и тд. Но главное запомнить основное правило: после замены не должно оставаться других переменных.
